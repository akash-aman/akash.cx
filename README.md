## üìñ About

**akash.cx** is a personal portfolio and blog built with the cutting-edge web technologies. It serves as a showcase for projects, a platform for sharing knowledge through technical blogs, and a hub for educational courses.

### Key Features
- **PWA Ready**: Installable as a native-like app on devices.
- **Markdown-Powered Blog**: Writes detailed articles with GFM, math support (Katex), and syntax highlighting.
- **Interactive Timeline**: Visualizes career journey and projects.
- **Dynamic Projects**: Showcases work with a tech-stack integrated grid.
- **Performance First**: Built on Next.js 16 with optimized static and server rendering.

---

## üìö Courses & Education

A dedicated section for structured learning paths, offering deep dives into:
- **Data Structures & Algorithms**: Efficient coding practices (e.g., LeetCode solutions).
- **Web Development**: Advanced techniques in React, Next.js, and modern CSS.

## üõ† Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Directory)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) & SASS
- **Data Fetching**: [GraphQL](https://graphql.org/) & [Apollo Client](https://www.apollographql.com/docs/react/)
- **Content**: MDX/Markdown with `remark` & `rehype` ecosystem
    - Math support via `rehype-katex`
    - Syntax highlighting via `react-syntax-highlighter`
    - GFM support via `remark-gfm`
- **PWA**: `@ducanh2912/next-pwa`
